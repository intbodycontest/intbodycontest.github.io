    How do I find my bank's SWIFT code?show/hide

    Many banks use SWIFT codes to ensure cross-border transactions are safe and reliable.

    This means you will need to know your bank's SWIFT code to facilitate these international funds transfers.

    To find your code please download the Standard Chartered SWIFT codes.

    How do I find the Local Branch code for my branch?show/hide

    If funds are being sent to you from within your country, you may need to know the code for your branch. To find your branch code please download the Standard Chartered bank and branch codes.
