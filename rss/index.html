    What is RSS? 

    RSS (Really Simple Syndication) is a web format that allows you to receive the latest content from a website, with ease, into your own browser, email inbox.

    You can get the latest updates and video in one place, as soon as it's published, without having to visit us directly.

    What software do I need to use RSS?

    You’ll need an RSS feed reader to get RSS. You can either use a web-based reader or a desktop news reader.

    • A web based reader is useful if you want to access feeds from multiple computers or devices
    • Alternatively you can use dedicated reader software which you will need to download and install on your computer
    • Some email software also allows you to read RSS feeds. 

    Most browsers support RSS and you can find many types of readers available on the internet. Some are free to download.
